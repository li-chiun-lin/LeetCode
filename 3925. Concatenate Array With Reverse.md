# Easy

You are given an integer array $nums$ of length $n$.

Construct a new array $ans$ of length $2 \times n$ such that the first $n$ elements are the same as $nums$, and the next $n$ elements are the elements of $nums$ in reverse order.

Formally, for $0 \leq i \leq n - 1$:

- $ans[i] = nums[i]$
- $ans[i + n] = nums[n - i - 1]$

Return an integer array $ans$.

```cpp
class Solution {
public:
    vector<int> concatWithReverse(vector<int>& nums) {
        for (int i = size(nums) - 1; i >= 0; --i)
            nums.push_back(nums[i]);

        return nums;
    }
};
```
