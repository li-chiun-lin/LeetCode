# Medium

You are given an integer array $nums$.

Return the length of the longest subsequence in $nums$ whose bitwise XOR is non-zero. If no such subsequence exists, return $0$.

```cpp
class Solution {
public:
    int longestSubsequence(vector<int>& nums) {
        int n = size(nums);
        int cnt0 = 0;
        int val = 0;

        for (auto x : nums)
        {
            if (x == 0)
                ++ cnt0;

            val ^= x;
        }

        if (cnt0 == n)
            return 0;

        if (val)
            return n;
        else
            return n - 1;
    }
};
```
