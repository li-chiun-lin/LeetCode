# Easy

You are given a string $s$ consisting only of lowercase English letters.

A prefix of $s$ is called a residue if the number of distinct characters in the prefix is equal to $len(prefix) \mod 3$.

Return the count of residue prefixes in $s$.

```cpp
class Solution {
public:
    int residuePrefixes(string s) {
        set<char> ss;
        int ans = 0;
        int len = 0;

        for (char c : s)
        {
            ss.insert(c);
            ++ len;

            if (size(ss) == len % 3)
                ++ ans;
        }

        return ans;
    }
};
```
