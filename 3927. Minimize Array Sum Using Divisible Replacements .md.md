# Medium

You are given an integer array $nums$.

You can perform the following operation any number of times:

- Choose two indices $a$ and $b$ such that $nums[a] \mod nums[b] == 0$.
- Replace $nums[a]$ with $nums[b]$.

Return the minimum possible sum of the array after performing any number of operations.

```cpp
class Solution {
public:
    long long minArraySum(vector<int>& nums) {
        sort(begin(nums), end(nums));
        int n = size(nums);
        long long sum = 0;
        queue<int> que(begin(nums), end(nums));

        while (int n = size(que))
        {
            int x = que.front();
            que.pop();
            sum += x;

            for (int i = 1; i < n; ++i)
            {
                if (que.front() % x == 0)
                    sum += x;
                else
                    que.push(que.front());

                que.pop();
            }
        }

        return sum;
    }
};
```
