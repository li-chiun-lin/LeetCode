# Easy

You are given an integer array $nums$.

You replace each element in $nums$ with the sum of its digits.

Return the minimum element in $nums$ after all replacements.

```cpp
class Solution {
public:
    int minElement(vector<int>& nums) {
        int ans = INT_MAX;

        for (int x : nums)
        {
            int s = 0;

            while (x)
            {
                s += x % 10;
                x /= 10;
            }

            ans = min(ans, s);
        }

        return ans;
    }
};
```
