# Medium

You are given an integer array $nums$.

You need to remove exactly one prefix (possibly empty) from $nums$.

Return an integer denoting the minimum length of the removed such that the remaining array is strictly increasing.

```cpp
class Solution {
public:
    int minimumPrefixLength(vector<int>& nums) {
        int n = size(nums);

        for (int i = n - 2; i >= 0; --i)
            if (nums[i] >= nums[i +  1])
                return i + 1;

        return 0;
    }
};
```
