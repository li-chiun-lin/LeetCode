# Mediuum

You are given a string $s$ of length $n$ consisting of lowercase English letters.

You must perform exactly one operation by choosing any integer $k$ such that $1 \leq k \leq n$ and either:

- reverse the first $k$ characters of $s$, or
- reverse the last $k$ characters of $s$.

Return the lexicographically smallest string that can be obtained after exactly one such operation.

```cpp
class Solution {
public:
    // Return lexicographically smallest string obtainable by exactly one operation:
    // choose 1 <= k <= n and either reverse first k chars or reverse last k chars.
    string lexSmallest(string s) {
        int n = s.size();
        // best will hold the best (smallest) candidate found so far.
        // initialize with a valid "one operation" result: reverse first 1 (no change but counts).
        string best = s; // we'll update with real operations below

        // Try all prefix reversals: reverse first k characters (k = 1..n)
        for (int k = 1; k <= n; ++k) {
            string t = s;
            reverse(t.begin(), t.begin() + k); // reverse the prefix [0..k-1]
            if (t < best) best = t;
        }

        // Try all suffix reversals: reverse last k characters (k = 1..n)
        for (int k = 1; k <= n; ++k) {
            string t = s;
            reverse(t.begin() + (n - k), t.end()); // reverse suffix [n-k..n-1]
            if (t < best) best = t;
        }

        return best;
    }
};
```
