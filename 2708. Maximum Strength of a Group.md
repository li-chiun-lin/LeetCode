# Medium

You are given a 0-indexed integer array $nums$ representing the score of students in an exam. The teacher would like to form one non-empty group of students with maximal strength, where the strength of a group of students of indices $i_0, i_1, i_2, ... , i_k$ is defined as $nums[i_0] \times nums[i_1] \times nums[i_2] \times ... \times nums[i_kâ€‹]$.

Return the maximum strength of a group the teacher can create.

```cpp
class Solution {
public:
    long long maxStrength(vector<int>& nums) {
        vector<int> neg;
        long long ret = 1;
        int cnt_pos = 0;
        int cnt_zero = 0;

        if (nums.size() == 1)
            return nums[0];

        for (int x : nums)
        {
            if (x > 0)
            {
                ret *= x;
                ++ cnt_pos;
            }
            else if (x < 0)
                neg.push_back(-x);
            else
                ++ cnt_zero;
        }

        if (cnt_pos == 0 && neg.size() < 2)
            return 0;

        sort(rbegin(neg), rend(neg));

        if (neg.size() % 2)
            neg.pop_back();

        for (auto x : neg)
            ret *= x;
        
        return ret;
    }
};
```
