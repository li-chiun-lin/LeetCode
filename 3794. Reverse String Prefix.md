# Easy

You are given a string $s$ and an integer $k$.

Reverse the first $k$ characters of $s$ and return the resulting string.

```cpp
class Solution {
public:
    string reversePrefix(string s, int k) {
        int n = size(s);

        for (int i = 0, j = k - 1; i < j; ++i, --j)
            swap(s[i], s[j]);

        return s;
    }
};
```
