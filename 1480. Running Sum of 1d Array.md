# Easy

Given an array $nums$. We define a running sum of an array as $runningSum[i] = sum(nums[0], â€¦, nums[i])$.

Return the running sum of $nums$.

```cpp
class Solution {
public:
    vector<int> runningSum(vector<int>& nums) {
        int n = nums.size();
        vector<int> ret(n);
        
        ret[0] = nums[0];
        
        for (int i = 1; i < n; ++i)
            ret[i] = ret[i - 1] + nums[i];
            
        return ret;
    }
};
```
