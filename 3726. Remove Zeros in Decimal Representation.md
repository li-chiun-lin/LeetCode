# Easy

You are given a positive integer $n$.

Return the integer obtained by removing all zeros from the decimal representation of $n$.

```cpp
class Solution {
public:
    long long removeZeros(long long n) {
        long long base = 1;
        long long m = 0;

        while (n)
        {
            int d = n % 10;

            if (d)
            {
                m += base * d;
                base *= 10;
            }

            n /= 10;
        }

        return m;
    }
};
```
