# Medium

You are given an integer $n$.

Return the total number of commas used when writing all integers from $[1, n]$ (inclusive) in standard number formatting.

```cpp
class Solution {
public:
    long long countCommas(long long n) {
        long long ans = 0;
        long long val = 1000;

        while (val <= n)
        {
            ans += n - val + 1;
            val *= 1000;
        }

        return ans;
    }
};
```
