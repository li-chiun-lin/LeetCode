# Medium

You are given an integer `n`.

An integer `x` is considered **good** if there exist **at least** two **distinct** pairs `(a, b)` such that:

- `a` and `b` are positive integers.
- `a <= b`
- `x = a3 + b3`

Return an array containing all good integers **less than or equal to** `n`, sorted in ascending order.

```cpp
class Solution {
public:
    vector<int> findGoodIntegers(int n) {
        vector<int> cube;
        int i = 1;
        long long i3 = 1;

        while (i3 < n)
        {
            cube.push_back(i3);
            ++ i;
            i3 = i * i * i;
        }

        int m = size(cube);
        map<int, int> freq;
        vector<int> ans;

        for (int i = 0; i < m; ++i)
            for (int j = 0; j <= i && cube[i] + cube[j] <= n; ++j)
                ++ freq[cube[i] + cube[j]];

        for (auto [k, v] : freq)
            if (v >= 2)
                ans.push_back(k);

        sort(begin(ans), end(ans));

        return ans;
    }
};
```
