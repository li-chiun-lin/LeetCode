# Medium

You are given an integer array `nums`.

An array is considered **alternating prime** if:

- Elements at **even** indices (0-based) are **prime** numbers.
- Elements at **odd** indices are **non-prime** numbers.

In one operation, you may **increment** any element by 1.

Return the **minimum** number of operations required to transform `nums` into an **alternating prime** array.

```cpp
class Solution {
public:
    int minOperations(vector<int>& nums) {
        int ub = *max_element(begin(nums), end(nums)) * 10;
        vector<long long> prime;
        vector<bool> sieve(ub, true);

        sieve[0] = sieve[1] = false;

        for (long long i = 2; i < ub; ++i)
            if (sieve[i])
            {
                prime.push_back(i);
                
                for (long long j = i * i; j < ub; j += i)
                    sieve[j] = false;
            }

        int n = size(nums);
        int ans = 0;

        for (int i = 0; i < n; ++i)
        {
            if (i % 2 == 0)
            {
                if (! sieve[nums[i]])
                {
                    auto nxt = lower_bound(begin(prime), end(prime), nums[i]);
                    ans += *nxt - nums[i];
                }
            }
            else
            {
                if (sieve[nums[i]])
                {
                    if (nums[i] == 2)
                        ans += 2;
                    else 
                        ++ ans;
                }
            }
        }

        return ans;
    }
};
```
